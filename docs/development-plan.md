## Рекомендация по порядку разработки

Начать с Фаз 1-3 для быстрого создания работающего прототипа. Это даст базовый функционал для ручной проверки API. Затем добавлять функции постепенно, тестируя каждую фазу перед переходом к следующей. Нейросеть оставить на самый конец, когда весь остальной функционал будет стабильно работать.
