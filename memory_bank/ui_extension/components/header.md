# Header Component

## Source
`components/Header.tsx`

## Role
Provides top-level navigation and dashboard tab switching.

## Responsibilities
- Show the current active tab.
- Let the user switch workspace sections.
- Persist the selected dashboard tab in the URL query string.
- Keep tab navigation reachable on compact widths via horizontal scrolling.
- Provide the user-facing light/dark theme toggle.
- Support the dashboard shell while child content changes underneath.
